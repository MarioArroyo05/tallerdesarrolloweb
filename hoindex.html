Curso de Herramientas Ofimaticas

Aprenderas a utlizar nuevas herraminetas que te ayudan en tu entorno de trabajo.

Descripcion

Un curso de ofimática es un programa de aprendizaje diseñado para capacitar a los usuarios en el uso de herramientas informáticas relacionadas con tareas de oficina. Estos cursos suelen cubrir una variedad de programas y aplicaciones, como Microsoft Word, Excel, PowerPoint, Access y Outlook.
Los contenidos de un curso de ofimática pueden incluir:
* Sistema operativo Windows.
* Editor de textos Microsoft Word.
* Hojas de cálculo con Microsoft Excel.
* Bases de datos con Microsoft Access.
* Presentaciones en Microsoft PowerPoint.
* Gestión y organización de correo electrónico con Outlook.

OBJETIVO DEL CURSO

- Familiarícese con las principales herramientas de ofimática, como procesadores de texto, hojas de cálculo, programas de presentación y gestores de correo electrónico.
- Aprender a utilizar las funciones básicas y avanzadas de las herramientas de ofimática para crear, editar y formatear documentos, hojas de cálculo y presentaciones.

TEMAS DEL CURSO A ABORDAR:

* Introducción a la informática y conceptos básicos.
* Uso del sistema operativo Windows.
* Procesador de texto (por ejemplo, Microsoft Word): creación, edición y formato de documentos, uso de estilos, tablas, imágenes y herramientas de revisión.
